Fiche projet — Harry Potter

Le Monde de Harry Potter

Site thématique réalisé en HTML, CSS, JavaScript — Intégration d'une API.
Voir le site
Ce projet est un site thématique entièrement codé à la main (HTML/CSS/JS) autour de l'univers de Harry Potter. L'objectif était d'utiliser une API pour récupérer dynamiquement des données (personnages, sorts, écoles) et les afficher avec une interface moderne.

Architecture & Objectifs

Composant Description
Consommation d'API Requêtes fetch() asynchrones pour récupérer des données JSON externes et les intégrer dans les différentes pages du site.
HTML / CSS / JavaScript Construction d’une interface moderne : structure HTML, mise en forme CSS, interactivité via JavaScript (modals, filtres, interactions dynamiques).
